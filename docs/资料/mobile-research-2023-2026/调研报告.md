# 2023-2026 移动端逆向趋势调研报告

> 调研日期: 2026-06-30
> 来源标注: 已验证的标注版本号/日期；领域通识标注【待核对】

## 一、趋势概览

1. **跨平台框架成主流载体**：Flutter/RN(Hermes)/KMP/MAUI 应用大量出现，"反编译 DEX → 读 Java"链失效
2. **Frida 17.x 升级为系统级观测平台**（17.15.3, 2026-06-22）：frida-strace、eBPF spawn gater、CFG API、iOS 26/非JB支持、ptrace-free 注入
3. **iOS 17/18/26 防护层叠**：PPL/MIE 使传统注入收窄；Dopamine 越狱适配
4. **Android 16（API 36, GKI 6.6）**：kCFI/PAC/MTE/SCS 全启用；Play Integrity 收紧自动化
5. **国产加固持续演化**：DEX VMP、native 化反调试加强

## 二、关键技术清单

### 技术 1 — Flutter Dart AOT 静态逆向（blutter）
- **频率**: 高
- **原理**: 编译对应版本 Dart VM 运行时，解析 snapshot，恢复类/函数/对象池
- **步骤**: `python3 blutter.py lib/arm64-v8a out_dir` → 产出 asm/* + blutter_frida.js + objs.txt
- **工具**: blutter (worawit/blutter, 2.5k★, 2026-04 更新)
- **局限**: **仅 Android arm64，iOS 不支持**

### 技术 2 — Flutter SSL Pinning Bypass（reFlutter）
- **频率**: 高
- **变化**（已验证）: **Flutter 3.24.0+ 移除硬编码代理 IP/端口**，需设备级 `adb shell settings put global http_proxy`
- **工具**: Impact-I/reFlutter (2.7k★, 2026-06 活跃)；原 ptswarm 版已归档

### 技术 3 — React Native Hermes 字节码逆向（hermes-dec）
- **频率**: 中高
- **原理**: RN 0.70+ 编译为 HBC；hermes-dec 解析头 → 反汇编 → 伪 JS
- **工具**: P1sec/hermes-dec (v0.1.5, 支持 bytecode v98, 2026-06-24)
- **局限**: 反编译不还原循环/条件

### 技术 4 — iOS PPL/MIE 对抗
- **频率**: 中
- **原理**: PPL 保护页表/代码签名，限制运行时修改代码页
- **解题侧**: 依赖 Dopamine 越狱 + frida-server 17.x（处理 arm64e PAC、rootless、page plan）
- **来源**: 8kSec MIE Deep Dive (2026-01/02)；PPL 内部细节【待核对】

### 技术 5 — Android 16 SELinux 分析
- **频率**: 中（高难度 Android 题/内核利用题）
- **关键**: setools-android dump 实时策略；**neverallow 仅编译期检查，运行时不强制**——magiskpolicy 可自由添加
- **vendor 域策略最薄弱**
- **来源**: 8kSec Android SELinux Internals Part IV (2026-06-24)

### 技术 6 — 反 Frida 绕过（frida-strace 观测法）
- **频率**: 高
- **新方法**: Frida 17.8.0+ 的 `frida-strace` 观测应用发起的 syscall → 定位检测点 → 针对性 hook
- **ptrace 被占用**: Frida 17.11.0 起 ptrace-free 注入回退（走 /proc/$pid/mem）
- **stealth hook**: 自定义 redirect emitter 降低可预测性

### 技术 7 — DEX VMP / 国产加固对抗 【部分待核对】
- **通用思路**: 先脱壳提取真实 DEX（FART/BlackDex/frida-dexdump）→ VMP 部分需 dump 执行轨迹还原语义
- **最新版本特征**: 【待核对——需实际样本验证】

### 技术 8 — jadx 生态化
- **版本**: 1.5.5 (2026-02, 49.2k★)
- **新功能**: --deobf 混淆还原、kotlin-metadata 还原、--call-graph 调用图导出、mappings 加载

## 四、解题方法论

1. **指纹识别**: Flutter? libapp.so+libflutter.so；RN? Hermes 头；MAUI? libmonodroid.so；加固? so 名/特征
2. **按框架选工具**: Flutter→blutter；RN→hermes-dec；原生→jadx(+脱壳)；iOS→ipsw+Frida
3. **流量优先**: 能用 reFlutter/objection 抓流量的先走流量
4. **反 Frida**: frida-strace 观测 syscall → 定位检测 → stealth hook
5. **iOS 先确认版本**: 17+ 依赖 Dopamine + frida 17.x
