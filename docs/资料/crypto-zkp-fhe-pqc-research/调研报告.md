# ZKP / FHE / PQC — CTF 调研报告（2023–2026）

> 调研日期: 2026-06-30
> 来源标注: ✅ = 一手验证；📚 = 社区共识（待核对题名）；⚠️ = 待核对

## 一、趋势概览

1. 2022 年 Castryck-Decru 攻击后，SIDH/SIKE 类题沦为"跑现成脚本"类别。
2. NIST PQC 落定后，CTF 出题重心从"破方案"转向"破实现/破参数"（模数过小用 LLL、FO 变换错配、NTT 缺陷）。
3. ZKP 题随 circom/snarkjs/halo2 普及而井喷，**under-constrained circuit** 与 **Fiat-Shamir/Frozen Heart** 是两大母题。
4. FHE 题套路收敛：噪声预算过大、解密预言机滥用、LWE/RLWE 参数不足。

## 二、关键技术清单

### 技术 1 — Fiat-Shamir / Frozen Heart（伪造证明）✅
- **出现频率**: 极高（ZKP 母题）
- **核心原理**: 交互式 Σ 协议中 challenge `c = H(transcript)`。若哈希输入缺关键公开量（漏掉公钥 h），证明不绑定 statement。
- **关键步骤**（ZKDocs）:
  1. 任取 u、随机 z
  2. `c = H(g,q,u)`（用题里缺漏的输入集）
  3. 反推 `h = (g^z / u)^{1/c}`
  4. 提交 `(u, c, z)` → 伪造成功
- **来源**: ✅ ZKDocs https://www.zkdocs.com/docs/zkdocs/protocol-primitives/fiat-shamir/

### 技术 2 — HVZKP 被恶意验证者利用 ✅
- **出现频率**: 中高（"证明你知道 N 的分解"类）
- **核心原理**: 交互协议假设 honest verifier，恶意验证者用精心 challenge 反推密钥。
- **Two-Prime-Divisor**: 发 `ρ_i = r_i² mod N`，prover 返回 `σ_i`；当 `σ_i ≠ ±r_i`，`gcd(N, σ_i − r_i)` 给出因子
- **Short Factoring Proof**: `e = A`，由 `y = r + (N−φ(N))·e` 反解 `φ(N)`
- **来源**: ✅ ZKDocs https://www.zkdocs.com/docs/zkdocs/security-of-zkps/when-to-use-hvzk/

### 技术 3 — Under-Constrained Circuit 📚
- **出现频率**: 极高（circom/snarkjs/PLONK 母题）
- **关键**: 审 `.circom` 找未约束信号（`<--` 赋值而无 `<==` 约束），构造满足等式但非预期值的 witness

### 技术 4 — Castryck-Decru SIDH 密钥恢复 ✅
- **出现频率**: 高（SIDH/SIKE 题的"一键"解法）
- **关键**: 构造积曲面，Richelot (2,2)-isogeny 分裂逐位恢复私钥
- **Sage 性能**: SIKEp434≈10min, baby p64<10s
- **来源**: ✅ https://github.com/GiacomoPope/Castryck-Decru-SageMath

### 技术 5 — 格基归约攻 LWE/RLWE 📚
- 构造格 → `matrix.LLL()` / BKZ → 恢复 s 或 e

### 技术 6 — Dilithium nonce 泄漏 📚
- `z = y + c·s`，y 重用/低熵 → 多组签名 LLL 恢复 s

### 技术 7 — FHE 噪声/解密预言机滥用 📚
- BFV/BGV 噪声爆炸；CKKS 近似同态密钥相关噪声；oracle 查询收集 (c,m) 对 → LWE 归约

## 四、解题方法论

1. **定类**: 泄露辅助点→同态；给 oracle→代数/格；给电路/证明→ZK 实现层
2. **查现成攻击**: 同态类查 eprint 2022/975；ZK 类查 ZKDocs
3. **缩参重跑**: 套开源脚本先跑通再迁移
4. **审输入边界**: Fiat-Shamir 看哈希全集；HVZKP 看谁能控制 challenge
5. **格化**: 凡 LWE/RLWE/nonce 统一构造格 + LLL

## 五、知识库缺失识别

2. 缺 FHE CTF 参数→攻击对照表
3. 缺 PQC FO 变换缺陷专题
4. 缺 SIDH 攻击复现脚本本地化（含 Sage 版本踩坑）
